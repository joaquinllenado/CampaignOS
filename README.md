# Nozo Hack

Bun workspace with an Elysia API for backend and agent logic, plus a React + Tailwind CSS frontend.

## Getting Started

```bash
bun install
bun run dev
```

- API: `http://localhost:3001`
- Web: `http://localhost:5173`

## Scripts

```bash
bun run dev
bun run typecheck
bun run build
```

## Environment

Copy `.env.example` into `.env` and adjust values as needed.
